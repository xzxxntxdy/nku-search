# 开源许可与引用说明

本项目 clone 了多个成熟搜索和 Web 框架作为参考。`references/` 不进入最终提交包，项目代码为针对作业需求的重新实现或依赖调用。

| 框架 | 许可证/说明 | 本项目使用方式 |
|---|---|---|
| Scrapy | BSD License | 仿照 settings/items/pipelines/spiders 结构，使用 pip 依赖运行爬虫 |
| Whoosh | BSD License | 仿照 schema/query/scoring/searcher 分层重新实现本地 BM25F |
| FastAPI Full Stack Template | MIT License | 仿照 API-first 前后端分离组织方式 |
| Ant Design Pro | MIT License | 仿照控制台布局和企业级页面组织方式，使用 Ant Design 依赖 |
| Elastic Search UI | Apache 2.0 | clone 后参考 connector、facet、paging 的分层结构，最终以本项目代码改写实现 |
| Haystack | Apache 2.0 | 仿照 component pipeline，重新实现 SearchPipeline |
| Meilisearch | MIT AND BUSL-1.1 | 仅作为搜索引擎任务/索引设置参考，不复制源码 |

合规处理：

- 不把 `references/` 打包进提交压缩包。
- 不直接搬运大段源码作为自己的实现。
- 对可复用框架采用依赖方式调用，例如 Scrapy、FastAPI、Ant Design；对 Search UI 前端结构采用参考后改写方式，避免运行时 Provider 循环触发。
- 对搜索引擎内核采用重新实现方式，保留模块命名和文档说明用于解释参考来源。

新增关键实现：

- `nku_search/crawler_project/`：Scrapy 项目化爬虫。
- `nku_search/schema.py`、`scoring.py`、`local_index.py`：Whoosh 风格 BM25F 检索内核。
- `nku_search/search_pipeline.py`：Haystack 风格检索 pipeline。
- `frontend/src/SearchConsole.tsx`：React + Ant Design 搜索控制台，集中实现 facet、paging、suggestion、document search、history、account 和 click tracking。
