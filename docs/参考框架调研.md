# 参考框架调研与仿照说明

本项目 clone 了多个成熟项目到 `references/`，只用于阅读和对照，最终提交包会排除该目录。

## Scrapy

地址：`https://github.com/scrapy/scrapy`

参考点：`scrapy.cfg`、`items.py`、`pipelines.py`、`settings.py`、`CrawlSpider + Rule + LinkExtractor`、`JOBDIR` 去重与断点续爬。

本项目对应：`scrapy.cfg`、`nku_search/crawler_project/settings.py`、`items.py`、`pipelines.py`、`spiders/nku_crawl.py`。

## Whoosh

地址：`https://github.com/whoosh-community/whoosh`

参考点：`fields.py`、`qparser/`、`query/`、`scoring.py`、`searching.py`。

本项目对应：`nku_search/schema.py`、`scoring.py`、`advanced_query.py`、`local_index.py`。

## FastAPI Full Stack Template

地址：`https://github.com/fastapi/full-stack-fastapi-template`

参考点：API-first 后端、前后端分离、登录/用户/JSON route 分层。

本项目对应：`nku_search/web.py` JSON API、FastAPI 托管 `frontend/dist` 的单端口部署、cookie session。

## Ant Design Pro

地址：`https://github.com/ant-design/ant-design-pro`

参考点：控制台布局、菜单导航、卡片、表格、列表、统计、诊断页。

本项目对应：`frontend/src/SearchConsole.tsx`。

## Elastic Search UI

地址：`https://github.com/elastic/search-ui`

参考点：`SearchProvider`、`SearchBox`、`Facet`、`PagingInfo`、`ResultsPerPage`、`Paging`、`APIConnector`。

本项目对应：`frontend/src/SearchConsole.tsx`。最初版本直接使用 Elastic Search UI npm 包；为避免 Provider 状态循环导致页面卡死，最终改写为同样的 connector / facet / paging / suggestion 分层结构，由 Ant Design 受控组件承载。

## Haystack

地址：`https://github.com/deepset-ai/haystack`

参考点：`Pipeline`、component graph、retriever/ranker/builder 分层。

本项目对应：`nku_search/search_pipeline.py`、`/api/pipeline`。

## Meilisearch

地址：`https://github.com/meilisearch/meilisearch`

参考点：search engine settings、task/status model、searchable/filterable/sortable attributes。

许可证包含 BUSL 部分，因此本项目只参考设计，不复制源码。
